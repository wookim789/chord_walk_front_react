# Chord wlak frontend

## 코드워크 프로젝트
#### 마스터 키보드 장비를 컴퓨터에 연결하면 웹브라우저(크롬)에서 연주
#### 피아노 연주시 키별 스케일 확인
#### 키별 다이어토닉 코드를 확인
<br/>

## 사용기술
1. react
2. nginx
3. github actions
4. aws codepiepline
5. aws s3
6. aws codedeploy
7. aws ec2
